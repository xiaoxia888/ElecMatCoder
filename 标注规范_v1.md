# 材料描述难度标注规范 v1

## 1. 标注目标
本标注用于判断“材料描述本身对抽取模型来说是简单还是困难”，不依据当前模型是否识别正确来标注。

每条样本只标：
- 难度：简单 / 困难
- 原因分类：无 / 表达复杂 / 语义竞争 / 材质复杂 / 锚点缺失
- 具体原因：对原因分类的简短说明
- 说明：1 句解释

## 2. 总体原则
### 2.1 简单
满足以下大多数条件时标简单：
- 主体明确直给
- 关键字段锚点清楚
- 没有明显复合/衬里/多材质
- 没有明显脏表达、黏连或工程冗余描述
- 不需要依赖行业知识去猜字段归属

### 2.2 困难
只要主要难点来自以下任一情况，即标困难：
- 写法脏、紧凑、黏连、重复或混写
- 主体或字段归属需要组合多个片段判断
- 材质是复合/衬里/多材质，明显增加理解成本
- 关键字段主要靠裸数字推断，锚点不足

## 3. 原因分类定义
### 3.1 表达复杂
当问题主要来自写法本身，而不是字段语义竞争时标此类。
典型情况：
- 黏连：如 BW20、3406WN、CL300RF
- 过度紧凑压缩：如 WN25(B)-16
- 标签被字符分割：如 E.F.W
- 同句夹杂大量工程说明或重复字段片段
- 壁厚体系混写：mm 与 SCH/STD/XS 同时出现

### 3.2 语义竞争
当问题主要来自“它到底属于哪个语义/字段”时标此类。
典型情况：
- 主体不是直给，需要靠多个片段组合判断，如 WOL-90 STD BWOlet
- 同一个值可能落到多个字段
- 紧凑工程写法中多个字段压缩在一起，需要语义拆分

### 3.3 材质复杂
当难点主要来自材质表达时标此类。
典型情况：
- 复合/衬里材质：PTFE/20#、S30408/PP、FRP/PVC
- 多材质并列
- 材质层次表达明显，理解成本高
注意：少见单一材质本身不直接算困难。

### 3.4 锚点缺失
当关键字段主要靠裸数字理解时标此类。
典型情况：
- 数值没有 DN/OD/THK/PN 等明确锚点，需要解释
- 裸数字可能在尺寸、壁厚、压力、材质之间竞争

## 4. 明确“不直接算困难”的情况
以下现象本身不直接构成困难：
- 异径规格本身，如 DN200x100、φ219.1xφ114.3
- 双尺寸/两段规格本身，只要表达规整
- 两段壁厚本身，只要体系一致、表达规整
- 多规范并列本身
- 少见种类或少见材质本身

## 5. 优先级规则
当一条样本同时满足多个原因时，按以下优先级确定主原因：
1. 材质复杂：如果复合/衬里/多材质是主要难点，优先标材质复杂
2. 表达复杂：如果问题主要来自黏连、重复、混写、紧凑表达，优先标表达复杂
3. 语义竞争：如果主要难在主体或字段归属判断，标语义竞争
4. 锚点缺失：如果主要靠裸数字推断，标锚点缺失

## 6. 同簇样本一致性规则
同一骨架、同一写法簇的样本，标签必须保持一致，除非存在明确语义差异。
例如：
- PIPE Lined ... PTFE/20# / S30408/PP -> 困难 / 材质复杂 / 复合衬里材质
- WOL-45 / WOL-90 ... BWOlet -> 困难 / 语义竞争 / Olet缩写主体需组合判断
- HALF COUPLING, SHCB, SW, FORGE, 20, ... -> 简单 / 无 / 无（主体已直给，不因 SHCB 或双尺寸直接判难）

## 7. 说明书写规则
- 说明只做解释，不引入新的标签概念
- “具体原因”应是稳定短语，不能一条一个说法
- 不允许把“说明”中的自由描述混入原因分类
