# 一阶段结构化标注草案-弯头

## 1. 目的

本文件只定义一件事：

- 弯头模型的一阶段结构化标注方式

这里的前提已经固定：

- 不同分类使用不同模型
- 一个模型一套结构

因此本文件只讨论：

- 弯头模型应该输出什么结构
- 弯头模型下 `BODY` 如何归一
- 弯头模型下 `TYPE` 哪些字段需要标注

---

## 2. 基本原则

### 2.1 一个模型一套结构

既然已经决定按分类拆模型，那么弯头模型只保留弯头真正需要的字段。

### 2.2 `BODY` 按编码种类归一

`BODY` 按**最终编码种类**归一。

原则：

- 编码相同，可以归一
- 编码不同，必须分开

当前弯头模型下，`BODY` 先统一标为：

- `弯头`

### 2.3 `TYPE` 字段标注分工

弯头模型下：

- `BODY`：标注弯头类编码种类
- `GEOMETRY.ANGLE`：标注角度
- `GEOMETRY.RADIUS`：标注长短半径
- `MANU`：标注制造方式
- `CONN`：标注连接方式
- `ENDS`：标注端部形式

例如：

- `90°承插焊弯头`
  - 应标成 `BODY=弯头`, `GEOMETRY.ANGLE=90`, `CONN=[SW]`

- `螺纹弯头`
  - 应标成 `BODY=弯头`, `CONN=[NPT]` 或 `CONN=[THD]`

- `长半径弯头`
  - 应标成 `BODY=弯头`, `GEOMETRY.RADIUS=LR`

- `短半径弯头`
  - 应标成 `BODY=弯头`, `GEOMETRY.RADIUS=SR`

## 3. 弯头模型的输出结构

当前建议弯头模型的一阶段输出中，`TYPE` 保留：

```json
"TYPE": {
  "BODY": "",
  "GEOMETRY": {
    "ANGLE": "",
    "RADIUS": ""
  },
  "MANU": [],
  "CONN": [],
  "ENDS": []
}
```

说明：

- `BODY`：弯头类编码种类
- `GEOMETRY.ANGLE`：角度
- `GEOMETRY.RADIUS`：半径类型
- `MANU`：制造方式
- `CONN`：连接方式
- `ENDS`：端部形式

## 4. 弯头模型下 `BODY` 当前建议

当前标准 `BODY`：

- `弯头`

### 4.1 `BODY` 完整归一化映射

以下为当前数据集中已出现写法到标准 `BODY` 的归一化结果。

- `ELBOW` -> `弯头`
- `Elbow` -> `弯头`
- `45DEG ELBOW` -> `弯头`
- `90DEG ELBOW` -> `弯头`
- `45度弯头` -> `弯头`
- `90度弯头` -> `弯头`
- `长半径弯头` -> `弯头`
- `短半径弯头` -> `弯头`
- `承插焊弯头` -> `弯头`
- `螺纹弯头` -> `弯头`

## 5. 弯头模型下其他 `TYPE` 字段

### 5.1 `GEOMETRY.ANGLE`

以下角度进入 `ANGLE`：

- `45`
- `90`
- `180`

原文出现明确角度数字时，直接提取数字。

### 5.2 `GEOMETRY.RADIUS`

以下半径类型进入 `RADIUS`：

- `LR`
- `SR`

归一化规则：

- `长半径` -> `LR`
- `短半径` -> `SR`
- `45E(L)` -> `ANGLE=45`, `RADIUS=LR`
- `90E(L)` -> `ANGLE=90`, `RADIUS=LR`
- `45E(S)` -> `ANGLE=45`, `RADIUS=SR`
- `90E(S)` -> `ANGLE=90`, `RADIUS=SR`

### 5.3 `MANU`

以下制造方式进入 `MANU`：

- `SMLS`
- `WELDED`
- `ERW`
- `EFW`
- `SAWL`
- `SAWH`
- `FORGED`

补充规则：

- 原文出现 `SEAMLESS` 时，统一标为 `SMLS`
- 原文出现 `无缝` 时，如需要标制造方式，也统一标为 `SMLS`

### 5.4 `CONN`

以下连接方式进入 `CONN`：

- `SW`
- `SF`
- `NPT`
- `THD`

### 5.5 `ENDS`

以下端部形式进入 `ENDS`：

- `FNPT`
- `MNPT`
- `MTE`
- `TSE`
- `FTE`
- `NPTF`

## 7. 当前结论

弯头模型当前采用以下规则：

- `TYPE` 保留：
  - `BODY`
  - `GEOMETRY`
  - `MANU`
  - `CONN`
  - `ENDS`
- `BODY` 当前统一为：
  - `弯头`
- `ANGLE` 标角度数字
- `RADIUS` 统一标 `LR` / `SR`
- `承插焊`
  - 通过 `CONN=[SW]` 承接
- `螺纹`
  - 通过 `CONN=[NPT]` 或 `CONN=[THD]` 承接

后续待补充：

- `180度弯头` 的真实样本例子
- `NPT` 与 `THD` 在弯头中的边界例子
- `Lateral` 是否完全排除出弯头模型的边界例子
