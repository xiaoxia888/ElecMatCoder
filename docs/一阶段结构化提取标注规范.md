# 一阶段结构化提取标注规范

## 总体原则

1. **原样提取**：从材料描述中提取信息时，原文是什么就提取什么，不做推断、不做同义词替换
2. **有限归一化**：大多数字段保持原文语义；`PRESSURE` 统一到固定格式，`MATERIAL` 的 `EXEC_STANDARD / GRADE / SPECIAL_REQ` 按本规范统一表达形式，但不做跨体系语义合并
3. **空值保留**：使用 keep 模式输出完整 JSON 骨架，无值字段保留空字符串或空数组

---

## JSON 结构总览

```json
{
  "TYPE": {
    "BODY": "",
    "GEOMETRY": { "ANGLE": "", "RADIUS": "" },
    "MANU": [],
    "CONN": [],
    "SEAL": [],
    "ENDS": []
  },
  "SIZE": { "DN": [], "OD": [], "INCH": [], "LENGTH": [] },
  "THICKNESS": { "MM": [], "SCHEDULE": [], "SERIES": [], "BWG": [], "INCH": [] },
  "PRESSURE": "",
  "MATERIAL": {
    "RELATION": "",
    "ITEMS": [
      { "EXEC_STANDARD": "", "GRADE": "", "SPECIAL_REQ": [] }
    ]
  },
  "STANDARD": [
    { "BODY": "", "GRADE": "", "METHOD": "", "APPENDIX": "" }
  ]
}
```

## 各字段标注规则

### 1. TYPE — 种类

#### 1.1 BODY（种类主体）

**原样提取描述中的种类名称**，保留原文的语言和表达形式，不做翻译或简化。

| 规则 | 示例 |
|------|------|
| 原文中文就标中文 | "90°弯头" → `"弯头"` |
| 原文英文就标英文 | "ELBOW DN100" → `"ELBOW"` |
| **中英文同时出现时优先选用中文** | "ELBOW 弯头" → `"弯头"`, "REDUCING TEE 异径三通" → `"异径三通"` |
| 去掉材质前缀（不锈钢/碳钢等） | "不锈钢承插焊支管座" → `"承插焊支管座"` |
| 去掉"中压""高压"等压力修饰 | "中压不锈钢管件90°弯头" → `"弯头"` |
| 去掉功能性修饰 | "输送流体用无缝钢管" → `"钢管"` |
| 种类被其他内容拆分时用 `;` 拼接 | "ELBOW DIN EN 10217-7-BENDING RADIUS 3D-90" → `"ELBOW;BENDING RADIUS 3D;90"` |
| 复合种类保留完整名称 | "带颈对焊法兰" → `"带颈对焊法兰"`, "HALF COUPLING" → `"HALF COUPLING"` |
| BL 等类型修饰保留 | "法兰盖 BL" → `"法兰盖 BL"` |

常见种类标注示例（中英文各自保留）：

| 中文标注 | 英文标注 |
|----------|----------|
| 弯头 | ELBOW / Elbow |
| 三通 / 等径三通 / 异径三通 | TEE / EQUAL TEE / REDUCING TEE |
| 同心异径管 / 偏心异径管 | CONCENTRIC REDUCER / ECC REDUCER |
| 管 / 钢管 / 管子 | PIPE / Pipe / TUBE |
| 法兰 / 带颈对焊法兰 | FLANGE / WN FLANGE |
| 管帽 / 封头 | CAP |
| 短节 / 短管 | NIPPLE / PIPE NIPPLE |
| 半管接头 | HALF COUPLING |
| 支管台 / 对焊管接台 | WELDOLET / SOCKOLET |
| 法兰盖 / 盲法兰 | BLIND FLANGE |
| 管箍 / 活接头 | COUPLING / UNION |
| 堵头 | PLUG |

#### 1.2 GEOMETRY（几何参数）

仅弯头类管件使用。

| 字段 | 规则 | 示例 |
|------|------|------|
| ANGLE | 提取角度数字 | "90°弯头" → `"90"`, "45度弯头" → `"45"` |
| RADIUS | 长半径=LR，短半径=SR | "长半径弯头" → `"LR"`, "SR" → `"SR"` |

**E(L)/E(S) 模式特殊处理：**

| 描述中的表达 | ANGLE | RADIUS |
|-------------|-------|--------|
| `45E(L)` | `"45"` | `"LR"` |
| `90E(L)` | `"90"` | `"LR"` |
| `45E(S)` | `"45"` | `"SR"` |
| `90E(S)` | `"90"` | `"SR"` |

绝对值半径（如 R=381mm）暂不提取。

#### 1.3 MANU（制造工艺）

**只标注参与编码的制造工艺**，SMLS（无缝）不参与编码，不标注。

| 标注值 | 含义 | 数据量 |
|--------|------|--------|
| WELDED | 焊接 | 327 |
| EFW | 电熔焊 | 287 |
| ERW | 电阻焊 | 144 |
| SAWL | 直缝埋弧焊 | 105 |
| SAWH | 螺旋埋弧焊 | 45 |
| SAW | 埋弧焊 | 1 |

注意：描述中出现"焊接方法：氩电连焊"等制造信息不属于本字段，不提取。

#### 1.4 CONN（连接方式）

| 标注值 | 含义 | 数据量 |
|--------|------|--------|
| SW | 承插焊（含 SCRD 归一化） | 402 |
| SF | 承插熔接 | 402 |
| NPT | 美标锥管螺纹 | 262 |
| THD | 螺纹连接 | 252 |

#### 1.5 SEAL（密封面形式）

| 标注值 | 含义 | 数据量 |
|--------|------|--------|
| RF | 突面 |  |
| FF | 全平面 | 48 |
| RTJ | 环连接面 | 44 |
| RJ | 环连接面 | 42 |
| FLRJ | 法兰环连接面 | 29 |
| Double-lip | 双唇密封面 |  |
| WAF/RF
| WAF/FF
| WAF/RJ
| MFM | 凹凸面 |  |
| M | 凸面 |  |
| F | 凹面 |  |

#### 1.6 ENDS（端部形式）

**只标注参与二阶段编码的端口型值**，不参与编码的（BLE、BE、PLE、PE）不标注。

注意：
- `SO`、`WN` 这类在法兰中直接决定主体子类型的代码，不进入 `ENDS`
- 例如 `SO FLANGE` 应直接标为 `BODY=带颈平焊法兰`
- 例如 `WN FLANGE` 应直接标为 `BODY=带颈对焊法兰`

| 标注值 | 含义 | 数据量 |
|--------|------|--------|
| FNPT | 内螺纹 | 175 |
| MNPT | 外螺纹 | 108 |
| MTE | 外螺纹端 | 31 |
| TSE | 螺纹小端 | 14 |
| NPTF | 干密封螺纹 | 10 |
| FTE | 内螺纹端 | 8 |

不标注的端部形式（不参与编码）：BLE（坡口大端）、BE（坡口端）、PLE（光管大端）、PE（光管端）。

两端不同时，数组中放多个独立值：`["FNPT", "MNPT"]`、`["TSE", "MNPT"]`。

### 2. SIZE — 尺寸

#### 2.1 DN（公称直径）

统一为 **DN+数字** 格式。

| 描述形式 | 标注 |
|----------|------|
| DN50 | `"DN50"` |
| DN 100 | `"DN100"` |
| 50NB | `"DN50"` |
| NPS14 | 转为 INCH `"14"` |

多尺寸场景（如异径管 DN150×100）：`["DN150", "DN100"]`

#### 2.2 OD（外径）

提取纯数字，不带单位。

| 描述形式 | 标注 |
|----------|------|
| φ159 | `"159"` |
| Φ89.0 | `"89"` |

#### 2.3 INCH（英制尺寸）

统一为**带 `"` 的格式**。

| 描述形式 | 标注 |
|----------|------|
| 3" | `'3"'` |
| 1-1/2" | `'1-1/2"'` |
| 1/2'' | `'1/2"'` |
| NPS14 | `'14"'` |

#### 2.4 LENGTH（长度）

统一以 **单位跟随原文描述**：原文带 mm 则保留 mm，原文不带则不加。

| 描述形式 | 标注 |
|----------|------|
| L=300mm | `"300mm"` |
| L=200（无单位） | `"200"` |
| 75 mm（短节长度） | `"75mm"` |
| LENGTH=85.0904 | `"85.0904"` |

#### 3.2 SCHEDULE

| 描述形式 | 标注 |
|----------|------|
| SCH40S | `"SCH40S"` |
| Sch.80 | `"SCH80"` |
| S-10S | `"SCH10S"` |
| S40 | `"SCH40"` |

#### 3.3 SERIES

| 标注值 | 含义 |
|--------|------|
| STD | 标准壁厚 |

#### 3.4 BWG

| 描述形式 | 标注 |
|----------|------|

#### 3.5 INCH（英制壁厚）

| 描述形式 | 标注 |

### 4. PRESSURE — 磅级 ⚠️ 唯一归一化字段

归一化为 **CL / LB / PN** 三种体系：

| 描述形式 | 标注 |
|----------|------|
| CL150, CL 150, CL.150 | `"CL150"` |
| CLASS150, Class150 | `"CL150"` |
| C150, C900 | `"CL150"`, `"CL900"` |
| 150LB, 150Lbs, 150# | `"150LB"` |
| PN16, PN 25 | `"PN16"`, `"PN25"` |
| PN2.5 | `"PN2.5"` |
| 16RF, 25RF | `"PN16"`, `"PN25"` |

注意：不同体系之间**不互转**（CL150 不转为 150LB，PN16 不转为 CL150）。

### 5. MATERIAL — 材质

#### 5.1 RELATION（材质关系）

| 值 | 含义 | 示例 |
|----|------|------|
| `single` | 单一材质 | 大多数情况 |
| `alternative` | 同一件材料的双认证、并列牌号或替代表达 | `"A106 Gr.B or A53 Gr.B"`、`"F316/F316L (Dual Certified)"`、`"304/304L"` |
| `composite` | 基材+衬里/覆层/复层等物理复合材质 | `"20/衬胶"`、`"20/PTFE"`、`"PTFE/20"` |

无材质信息时，RELATION 为空，ITEMS 为空数组。

判定原则：
- **不要只看是否有 `/`**，而是看两个材料是否同时物理存在于同一件上
- 若是同一件材料的双认证、双牌号、替代表达，标 `alternative`
- 若是基材与衬里/覆层/复层同时存在，标 `composite`
- 若只有一个明确材质，标 `single`

典型规则：
- `304/304L`、`F316/F316L (Dual Certified)`、`WP304/WP304L`、`S30408/S30403` → `alternative`
- `20/衬胶`、`20/PTFE`、`PTFE/20`、`CS/RPTFE`、`PTFE LINED` → `composite`
- 不要把 `PTFE`、`RPTFE`、`衬胶` 标到 `SPECIAL_REQ`

#### 5.2 ITEMS

每种材质一个 ITEM 对象：

##### EXEC_STANDARD（材质执行标准）

按**固定主写法**归一，只统一标准别名，不改变标准语义。

| 描述形式 | 标注 |
|----------|------|
| ASTM A403 / A403 | `"ASTM A403"` |
| ASTM A182 / A182 / SA182 | `"ASTM A182"` |
| ASTM A105 / A105 | `"ASTM A105"` |
| ASTM A105N | `"ASTM A105N"` |
| ASTM A106 / A106 / A.106 | `"ASTM A106"` |
| ASTM A53 / A53 / A.53 | `"ASTM A53"` |
| API5L | `"API 5L"` |
| GB/T14976 | `"GB/T14976"` |

常见统一规则：
- `A182 / ASTM A182 / SA182` → `ASTM A182`
- `A403 / ASTM A403` → `ASTM A403`
- `A312 / ASTM A312` → `ASTM A312`
- `A234 / ASTM A234` → `ASTM A234`
- `A105 / ASTM A105` → `ASTM A105`
- `A106 / A.106 / ASTM A106` → `ASTM A106`
- `A53 / A.53 / ASTM A53` → `ASTM A53`
- `A358 / ASTM A358` → `ASTM A358`
- `A269 / ASTM A269` → `ASTM A269`
- `A213 / ASTM A213` → `ASTM A213`
- `A516 / ASTM A516` → `ASTM A516`
- `A333 / ASTM A333` → `ASTM A333`
- `A216 / ASTM A216` → `ASTM A216`
- `API5L` → `API 5L`

注意：
- 仅统一同一标准的别名写法，不跨标准合并
- `ASTM A234M`、`ASTM A350M` 这类带 `M` 的标准按原样保留，不并入无 `M` 版本

##### GRADE（材质牌号）

按**统一表达形式**标注，保留材质语义差异，只统一大小写、下划线、`Gr.` 前缀和等级写法。

| 描述形式 | 标注 |
|----------|------|
| F304 | `"F304"` |
| SF304 | `"SF304"` |
| S30408 | `"S30408"` |
| 304 | `"304"` |
| 316L_Gr.II / 316L GR.II / 316LII | `"316L Gr.II"` |
| S32168I / S32168 GR.I / S32168_Gr.I | `"S32168 Gr.I"` |
| S32168III / S32168 GR.III / S32168_Gr.III | `"S32168 Gr.III"` |
| GR. B / GR.B / GRADE B | `"Gr.B"` |
| GRADE F304 / GR.F304 / Gr.F304 | `"Gr.F304"` |
| 20# | `"20#"` |
| 06Cr19Ni10 | `"06Cr19Ni10"` |
| WPB | `"WPB"` |
| CF415K | `"CF415K"` |

统一规则：
- `GRADE / GR. / Gr.` 前缀统一写成 `Gr.`
- 下划线去掉：`S30408_Gr.III` → `S30408 Gr.III`
- 粘连等级拆开：`S32168I` → `S32168 Gr.I`，`304III` → `304 Gr.III`
- 等级固定写成：`Gr.I` / `Gr.II` / `Gr.III`
- 大小写统一到常用业务写法，如 `S32168`、`TP316L`、`WP304`、`F316L`

注意：
- 只统一写法，不跨体系合并语义
- 例如 `S31603`、`316L`、`TP316L`、`WP316L`、`F316L` 必须保留差异，不能互相归并
- 复合材质不要压成单个 `GRADE`
- 例如 `20/PTFE`、`PTFE/20`、`20/衬胶` 不要标成单个 `GRADE`，应拆成多个 `ITEM`

##### SPECIAL_REQ（特殊要求）

**镀锌和 NACE 归一化处理**，其他按原文保留。

| 描述形式 | 含义 | 标注 |
|----------|------|------|
| NACE / NACE MR 0103 | 抗硫 | 归一化为 `"NACE"` |
| ANTI-H2S | 抗硫 | `"ANTI-H2S"` |
| CE | 抗硫 | `"CE"` |
| GALV / Galv. / 镀锌 / GALVANIZED / Zn / 热镀锌 等 | 镀锌 | 全部归一化为 `"GALV"` |

归一化说明：
- **镀锌**：原数据中存在 13+ 种写法（`Galv.`、`镀锌`、`GALVANIZED`、`Zn`、`Ga lv.` 等），全部归一化为 `"GALV"`（共 134 条）
- **NACE**：`NACE MR 0103` 等变体归一化为 `"NACE"`（共 9 条）
- `PTFE`、`RPTFE`、`衬胶`、`PTFE LINED` 这类属于材料组成或衬里信息，不属于 `SPECIAL_REQ`

### 6. STANDARD — 规范标准

每个标准一个对象，放入数组。

#### 6.1 BODY（标准主体）

原样提取标准号，**统一为无空格格式**（前缀紧跟编号）。

| 描述形式 | 标注 |
|----------|------|
| ASME B16.9 | `"ASME B16.9"` |
| GB/T12459 / GB/T 12459 | `"GB/T12459"` |
| HG/T20592 | `"HG/T20592"` |
| SH/T3408 | `"SH/T3408"` |
| NB/T47010 | `"NB/T47010"` |
| MSS SP-97 | `"MSS SP-97"` |
| EN 10216-5 / EN10216-5 | `"EN10216-5"` |
| DIN 17458 / DIN17458 | `"DIN17458"` |

格式化规则：
- **中国标准**（GB/T、SH/T、HG/T、NB/T、SY/T）：前缀紧跟编号，无空格，如 `GB/T12459`
- **欧洲标准**（EN、DIN）：前缀紧跟编号，无空格，如 `EN10216-5`、`DIN17458`
- **DIN EN 复合前缀**：保留中间空格，如 `DIN EN 10217-7`
- **ASME / MSS / JIS**：保持约定俗成的空格格式，如 `ASME B16.9`、`MSS SP-97`、`JIS B2220`
- **缺少前缀的简写**（`HG20553`、`GB12459`、`12459` 等）：原文如此，保持原样不补
- **年份后缀不提取**：4 位年份（如 `-2017`、`-2015`）不参与编码，从 BODY 中去除。如 `GB/T19326-2017` → `GB/T19326`

#### 6.2 GRADE（系列/等级）

| 描述形式 | 标注 |
|----------|------|
| GB/T12459(I系列) | BODY=`"GB/T12459"`, GRADE=`"I系列"` |
| GB/T12459(II) | BODY=`"GB/T12459"`, GRADE=`"II"` |
| HG/T20592(B) | BODY=`"HG/T20592"`, GRADE=`"B"` |
| HG/T20592(A) | BODY=`"HG/T20592"`, GRADE=`"A"` |
| Series I | GRADE=`"Series I"` |
| GB/T12771 TYPE V | GRADE=`"TYPE V"` |
| Ⅰa | GRADE=`"Ⅰa"` |

注意：括号内的系列/等级信息需拆分到 GRADE 字段，不能留在 BODY 中。

#### 6.3 METHOD（方法）

| 描述形式 | 标注 |
|----------|------|
| Method A / 方法A | `"Method A"` / `"方法A"` |
| Method E / 方法E | `"Method E"` / `"方法E"` |

#### 6.4 APPENDIX（附录）

| 描述形式 | 标注 |
|----------|------|
| HG/T20615(A) | BODY=`"HG/T20615"`, APPENDIX=`"A"` |
| 附录B | APPENDIX=`"附录B"` |

注意：`HG/T20615(A)` 中的 (A) 是**附录**不是系列，需正确区分 GRADE 和 APPENDIX。
