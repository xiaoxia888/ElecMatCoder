# 管道材料智能分词与编码方案

---

## 第一页：材料描述标签体系

### 管道材料描述的8类核心标签

| 序号 | 标签名称 | 英文代码 | 说明 | 示例 |
|:---:|:---|:---:|:---|:---|
| 1 | **种类** | TYPE | 产品基本类型 | 管子、法兰、弯头、三通 |
| 2 | **材质** | MATERIAL | 材料牌号 | 304、2205、A105、碳钢 |
| 3 | **尺寸** | SIZE | 尺寸规格 | DN100、φ108、SIZE:40 |
| 4 | **壁厚** | THICKNESS | 壁厚参数 | SCH40、4mm、S-10S |
| 5 | **磅级** | PRESSURE | 压力等级 | PN16、CL150、150LB |
| 6 | **规范** | STANDARD | 标准代码 | GB/T12459、HG/T20553(II) |
| 7 | **连接方式** | CONN | 安装连接方式 | BW、SW、对焊、WN、SO |
| 8 | **制造工艺** | MANU | 制造加工工艺 | 焊接、无缝、SMLS、锻制 |

### 标注示例

**原文**：`管子;SMLS;BE;304;GB/T14976;HG/T20553(II);φ18X3mm;SIZE:15`

**分词结果**：
- 管子 → **种类**
- SMLS → **工艺**
- 304 → **材质**
- GB/T14976、HG/T20553(II) → **规范**
- φ18 → **尺寸**
- 3mm → **壁厚**
- SIZE:15 → **尺寸**

## 第二页：分词方法对比

### 三种分词技术路线

**1. Jieba分词**
- 技术方案：基于词典的中文分词
- 优点：速度快、无需训练
- 缺点：无法识别专业实体，需大量自定义词典

**2. 大模型分词**
- 技术方案：LLM + Prompt Engineering
- 优点：泛化能力强、理解上下文
- 缺点：成本高、速度慢、不稳定

**3. BERT多任务模型**
- 技术方案：NER任务头 + 分类任务头
- 优点：速度快、准确率高、可控
- 缺点：需要标注数据训练

### 技术选型建议

```
                    准确性
                      ↑
                      │    ★ BERT多任务
                      │         (推荐)
                      │
                      │  ★ 大模型API
                      │
                      │
        ★ Jieba      │
        ─────────────┼──────────────→ 成本/速度
                     │
```

**结论**：推荐使用 **BERT多任务模型** 作为生产环境方案，大模型用于数据标注辅助。

## 第三页：BERT多任务模型实现

### 1. 整体思路

采用多任务学习框架，在一个BERT模型上同时训练两个任务：
- **NER任务**：序列标注，识别文本中的8类实体
- **分类任务**：文本分类，判断材料属于哪个大类

两个任务共享BERT编码器参数，通过联合训练相互促进。

### 2. 共享编码器（BERT）

- 使用 **bert-base-chinese** 预训练模型
- 输入材料描述文本，经过Tokenizer分词后输入BERT
- BERT输出每个token的768维向量表示，以及整个句子的[CLS]向量
- 这些向量同时供两个任务头使用

### 3. NER任务头

**作用**：对每个字符预测其实体标签

**结构**：
- 线性层：将BERT输出的768维向量映射到标签数量（17个BIO标签）
- CRF层：条件随机场，学习标签之间的转移概率，保证输出序列合法

**标签体系**（BIO格式）：
- O：非实体
- B-TYPE / I-TYPE：种类实体的开始/内部
- B-MATERIAL / I-MATERIAL：材质实体
- B-SIZE / I-SIZE：尺寸实体
- B-THICKNESS / I-THICKNESS：壁厚实体
- B-PRESSURE / I-PRESSURE：磅级实体
- B-STANDARD / I-STANDARD：规范实体
- B-CONN / I-CONN：连接方式实体
- B-MANU / I-MANU：制造工艺实体

### 4. 分类任务头

**作用**：判断整条材料描述属于哪个大类

**结构**：
- 取BERT输出的[CLS]向量（代表整个句子的语义）
- 经过Dropout + 全连接层，输出6分类结果

**分类类别**：管子、管件、法兰、螺栓、阀门、垫片

### 5. 训练策略

- **损失函数**：NER损失（CRF负对数似然） + 分类损失（交叉熵）
- **优化器**：AdamW，BERT层使用较小学习率（2e-5），任务头使用较大学习率
- **训练数据**：2000+ 条人工标注样本
